  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with CC-TS in minutes
  </p>
  
  <Steps>
    <Step title="Create your project">
      Use the interactive CLI to scaffold a new CC-TS project with all the configuration you need.
      
      ```bash
      bun create cc-ts my-project
      cd my-project
      ```
      
      Choose between a bundled app (complete with development server) or a library package for sharing code.
    </Step>
    
    <Step title="Install dependencies">
      CC-TS projects use standard package managers. Install your dependencies with your preferred tool.
      
      <CodeGroup>
      ```bash npm
      npm install
      ```
      
      ```bash bun
      bun install
      ```
      
      ```bash yarn
      yarn install
      ```
      </CodeGroup>
    </Step>
    
    <Step title="Start developing">
      Run the development server with watch mode and hot reload. Your TypeScript code will be compiled to Lua automatically.
      
      ```bash
      bun run dev
      ```
      
      The builder will watch your files, recompile on changes, and serve your Lua output over HTTP for easy in-game loading.
    </Step>
    
    <Step title="Build for production">
      When you're ready to deploy, build your project with optional minification.
      
      ```bash
      bun run build
      ```
      
      Your compiled Lua files will be output to the `dist` directory, ready to load into ComputerCraft.
    </Step>
  </Steps>
